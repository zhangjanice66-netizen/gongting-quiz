# 功能说明

本文档详细说明各功能模块。

## 验证码系统

### 验证码生成

- 长度：6位
- 字符集：大写字母（排除易混淆的字母）+ 数字
- 可选前缀：最多2个字符

### 验证码状态

| 状态 | 说明 |
|------|------|
| 未使用 | 新生成，从未输入 |
| 未提交 | 已输入验证码，正在答题 |
| 已提交 | 完成测试，答案已提交 |

### 管理员入口

- 入口位置：验证码页面
- 触发方式：连续点击「宫门准入」标题5次
- 登录后无需验证码，可直接进入后台

---

## 测试题目

- 总题数：12道
- 计分维度：A、B、C、D、E（5个维度）
- 每题根据选择累加对应维度分数

---

## 测试结果

### 结果类型

| 类型 | 维度 | 说明 |
|------|------|------|
| 权倾朝野型 | A | 管理能力突出 |
| 内卷达人型 | B | 勤奋努力型 |
| 实力超群型 | C | 专业能力强 |
| 社交高手型 | D | 人际交往能力强 |
| 降维打击型 | E | 综合能力强 |
| 平庸度日型 | - | 分数均衡 |

### 计算逻辑

取5个维度中得分最高的作为最终结果类型。

---

## 管理后台

### 功能模块

1. **数据概览**
   - 总测试次数
   - 活跃天数

2. **验证码查询**
   - 可查询单个验证码的完整生命周期

3. **验证码生成**
   - 自定义生成数量
   - 支持自定义前缀

4. **数据统计**
   - 按日期统计
   - 结果分布分析

5. **验证码列表**
   - 显示最近100条
   - 包含创建、输入、提交时间

---

## 安全性

### 管理员密钥

- 长度要求：至少16位
- 存储方式：数据库存储（明文）
- 使用方式：通过 HTTP 头传递

### API 防护

- 管理员接口需要密钥认证
- 验证码接口无防护（需配合 Nginx 访问控制）

---

## 注意事项

1. 每个验证码只能使用一次
2. 验证码输入后开始计时，提交后记录
3. 管理员密钥请妥善保管
4. 建议定期备份数据库文件
